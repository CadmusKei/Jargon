## Challenge 1 - Zero-to-Cluster Bootstrap

Provision a baseline HPC mini-cluster from clean images: 1 head node and 2 compute nodes (no switch). Secure remote access must be functional and documented. All nodes must be reachable according to your intended topology.

### Expected Deliverables

- Topology diagram (logical and host-level).
- Host inventory with node roles, IPs, and OS versions.
- SSH strategy (`ProxyJump` or equivalent) and key management notes.
- Baseline hardening checklist (users, SSH policy, package updates, service status).
- `results.md` table: node reachability, uptime, and essential service checks.
