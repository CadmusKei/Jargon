## Folders

| Folder | Description |
|---|---|
| `_Challenge_1` | Work and screenshots from Challenge 1 |
| `_Tutorial_1` | Work and screenshots from Tutorial 1 — one subfolder per member |
| `_Tutorial_2` | Work and screenshots from Tutorial 2 — one subfolder per member |
| `_Tutorial_3` | Work and screenshots from Tutorial 3 — one subfolder per member |
| `__Guides` | Step-by-step guides for current distro and software versions |
| `__Tutorial_Briefs` | Briefs and reference material for each tutorial topic |
| `Prep_Btop_From_Source` | Building and running Btop compiled from source — one subfolder per member |
| `Prep_Fastfetch` | Fastfetch configuration and output screenshots |
| `Prep_HPL_Install` | HPL benchmark installation process and screenshots |
| `Prep_HPL_Optimisation` | HPL tuning and optimisation results |
| `Prep_Passwordless_SSH` | Setting up passwordless SSH between nodes |
